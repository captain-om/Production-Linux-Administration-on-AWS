# Linux Commands Used

## System

```bash
hostnamectl
systemctl
journalctl
```

## Users

```bash
adduser
usermod
groups
id
```

## Networking

```bash
ip addr
ss -tulpn
ufw status
```

## Processes

```bash
ps aux
top
kill -9
```

## Nginx

```bash
nginx -t
systemctl restart nginx
tail -f /var/log/nginx/error.log
```

## Supervisor

```bash
supervisorctl status
supervisorctl reread
supervisorctl update
```
